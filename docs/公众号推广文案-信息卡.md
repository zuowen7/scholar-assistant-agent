## 定位
学术AI工作台，把论文的「读·想·写·审·排」五个环节收进一个桌面应用。核心理念是「写论文版的 Claude Code」——打开项目文件夹，Agent 直接操作文件，而不只是聊天。

## 五个核心功能

1. **读 — PDF翻译**
   - 5步流水线：parse → clean → chunk → translate → format
   - 17道文本清洗（去页眉页脚、修断行、合并跨页段落、还原连字符等）
   - 支持16种PDF格式，自动检测单栏/双栏
   - DeepL风格左右双栏对照 + 句子悬停高亮

2. **想 — 思维导图 + 论证地图**
   - 思维导图：Vue Flow画布 + AI自动扩展 + dagre自动布局
   - 论证地图：Toulmin结构（主张/依据/保证/支撑/限定/反驳），AI自动从原文提取

3. **写 — Monaco编辑器 + Agent工作区**
   - Agent有完整的文件操作工具：read_file / write_file / str_replace / grep_files / git_op
   - 打开项目文件夹后Agent直接在目录里工作
   - 访问工作区外的路径会触发审批弹窗
   - 编辑器有Ghost Text补全：打字后1.5秒自动出建议

4. **审 — 论证账本 + 对抗审稿（最独特的功能）**
   - 论证账本：自动从摘要/引言提取每条「承诺」，去正文逐条验证是否兑现
   - 每条锚定到精确字符位置，改稿后自动重定位（锚定/漂移/丢失三态）
   - Reviewer-2：模拟7种会议（NeurIPS/ICML/ICLR/ACL/CVPR/KDD/CHI）审稿风格
   - 支持rebuttal mini-chat，reviewer可以被说服
   - 三角度并行评审（method/experiment/writing）

5. **排 — 导出**
   - 6套LaTeX模板：IEEE Conf/IEEE Journal/ACM/NeurIPS/LNCS/Generic
   - Pandoc + Tectonic编译，支持PDF和Word导出

## 技术栈
- 前端：Vue 3 + TypeScript + Monaco Editor + Vue Flow
- 桌面壳：Tauri 2 (Rust，管理Python和Ollama进程)
- 后端：Python FastAPI，全链路SSE流式推送
- 本地LLM：Ollama + Qwen3:8b
- 云端LLM：21家（OpenAI/Anthropic/DeepSeek/Moonshot/智谱/通义千问/Gemini/火山方舟/百度千帆等）
- 向量库：ChromaDB + all-MiniLM-L6-v2
- 导出：Pandoc + Tectonic
- 所有组件打包进安装包，装完可离线使用

## 技术亮点（可选择性写）
- ReAct Agent手写的，没用LangChain/LlamaIndex
- 21家云端模型的tool-calling行为差异很大，做了统一抽象，每个模型有对应的工具调用指导
- DeepSeek返回tool_call时content字段不能是null（会400），做了适配
- Agent的纯文档问答不进多步循环，直接单次LLM流式回答，避免死循环
- 全局步数上限60，单工具累计超3次强制终止
