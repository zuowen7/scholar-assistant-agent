## 执行顺序建议

### 第 1 天：基础设施 + MCP
1. api_factory.py 拆分为 routers/ + services/ （0.5天）
2. MCP Server 支持 — 将翻译管道和 Agent 工具暴露为 MCP 协议（1-2天）
3. 翻译记忆库 TMCache 类 + ChromaDB 集成（1天）

### 第 2-3 天：核心翻译加速
4. 并行翻译 asyncio + Semaphore（1-2天）
5. 多轮精炼 refine=True 参数（1天，可与 #4 并行）

### 第 4-5 天：格式保留 + 用户体验
6. 双语 PDF 叠加 — Parser 保留坐标 + Formatter pdf_overlay 模式（3-5天，A 主攻）
7. 术语锚点系统后端 API + 前端交互（2-3天，A/B 并行）

### 第 6-7 天：打磨
8. 双语对比阅读 + Ollama 进程守护 + AI Panel diff

> **原则**：每次只做 1 件事，做完再切下一件。两个人各选一条互不阻塞的线并行。
