# Thesis Writing Standards

When writing or expanding academic theses:

1. **Structure**: Each section should have a clear topic sentence, 2-3 supporting paragraphs with evidence, and a transition to the next section.

2. **Argumentation**:
   - State claims explicitly ("We argue that...", "This paper demonstrates...")
   - Support each claim with evidence (citations, data, logical reasoning)
   - Address counterarguments where appropriate

3. **Academic Tone**:
   - Use formal, precise language
   - Avoid superlatives ("very", "extremely") — use quantifiable terms
   - Define technical terms on first use

4. **Citation Format**:
   - Inline: \cite{key} for narrative, \citep{key} for parenthetical
   - Every claim that references prior work must have a citation
   - Bibliography entries follow the paper's style (IEEE, ACM, APA)

5. **Figures and Tables**:
   - Every figure/table must be referenced in the text before it appears
   - Captions should describe both WHAT and WHY
   - Use \ref{fig:label} and \ref{tab:label} for cross-references
