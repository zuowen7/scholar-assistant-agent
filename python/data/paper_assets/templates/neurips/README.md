# NeurIPS Template

## Purpose
This folder stores the template assets for NeurIPS-style conference papers.

## Source
Official NeurIPS style files and template package.

## Typical Use
Used for papers submitted to NeurIPS and related experiments with machine learning conference formatting.

## Folder Contents
- `source/`: original template source files
- `sample_pdf/`: official sample PDF files
- `assets/`: images, bibliography files, and auxiliary assets
- `notes.md`: compilation notes and known issues

## Status
Partially collected. Official style file, sample source file, and sample PDF have been added.
