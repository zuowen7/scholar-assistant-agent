# Generic Article Template

## Purpose
This folder stores a minimal generic LaTeX article template for baseline paper-generation and compilation testing.

## Source
Locally created generic article template.

## Typical Use
Used as a fallback and baseline template for testing generated content without depending on venue-specific class files.

## Folder Contents
- `source/`: minimal LaTeX source files
- `sample_pdf/`: compiled example PDF files
- `assets/`: optional supporting assets
- `notes.md`: compilation notes and known issues
