# ACM Template

## Purpose
This folder stores the template assets for ACM-style conference or journal papers.

## Source
Official ACM template package.

## Typical Use
Used for ACM publications that require the ACM article or proceedings template.

## Folder Contents
- `source/`: original template source files
- `sample_pdf/`: official sample PDF files
- `assets/`: images, bibliography files, and auxiliary assets
- `notes.md`: compilation notes and known issues

## Status
Partially collected. Official ACM template files, sample PDFs, and supporting assets have been added.
