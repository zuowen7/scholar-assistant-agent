# IEEE Journal Template

## Purpose
This folder stores the template assets for IEEE-style journal papers.

## Source
Official IEEE journal template package.

## Typical Use
Used for journal submissions that require IEEE journal formatting.

## Folder Contents
- `source/`: original template source files
- `sample_pdf/`: official sample PDF files
- `assets/`: images, bibliography files, and auxiliary assets
- `notes.md`: compilation notes and known issues

## Status
Partially collected. Core LaTeX source files, sample PDFs, and basic assets have been added.
