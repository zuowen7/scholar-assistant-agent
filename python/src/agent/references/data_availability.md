# 数据可用性声明

使用本文件帮助用户准备期刊所需的数据可用性声明。

## 数据分类

| 类型 | 中文 | 访问路径 |
|------|------|---------|
| raw data | 原始数据 | 公共仓库 / 受控访问 |
| processed data | 处理后数据 | 公共仓库 / 文章附件 |
| source data | 源数据 | 仓库 / Supplementary |
| restricted data | 受限数据 | 需合理请求 |
| reused data | 复用公开数据 | 引用来源和编号 |

## 中→英 精确转换

- "可向通讯作者索取" → 过于模糊，需说明限制原因和申请流程
- "数据可用性声明" → "Data Availability"
- "合理请求" → "reasonable request"（需说明理由和审核路径）
- "补充材料" → "Supplementary Information"

## 弱表述 → 强表述

| 弱 | 强 |
|----|-----|
| available upon request | available in [Repository] under [DOI] |
| data not shown | Source data are provided as a Source Data file |
| 可向作者索取 | Qualified researchers may request access from [committee] at [contact] |

## 检查清单

- [ ] 每个支撑结论的数据集都有明确的存储位置
- [ ] 公开数据有 DOI/登录号/永久链接
- [ ] 受限数据说明了限制原因和申请路径
- [ ] 自己产生的数据和复用数据分开说明
- [ ] 数据集引用符合 DataCite 格式
