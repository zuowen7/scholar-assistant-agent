# 图表制作原则

使用本文件指导科研图表的创建和审核。

## Figure Contract（画图前先定）

1. 核心结论：图必须辩护的一句话主张
2. 证据链：每张 subpanel 对主张的贡献，删除无独立证据价值的 panel
3. 原型分类：quantitative grid / schematic-led composite / image plate + quant / asymmetric mixed-modality
4. 发表规格：尺寸、字体、可编辑文本、源数据、统计量、导出格式

## 排版规则

- 字体：Arial → DejaVu Sans → Liberation Sans
- 基础字号：7pt（论文），15-24pt（slides）
- SVG fonttype = none（保持文本可编辑）
- 背景白色（图表），黑色仅用于显微镜/体积渲染

## 配色原则

- 全图使用统一的方法族色系，而非最大色相分离
- 预留绿色/红色仅用于增益/下降/方向性信号
- 优先直接标注而非图例（当类别空间位置固定时）

## 面板布局

- 首选一个 hero panel + 从属证据面板
- 非对称布局 > 对称等分网格
- 图注 ≤ 300 词（Nature 风格）
- 标题 ≤ 75 字符（Nature 风格）
