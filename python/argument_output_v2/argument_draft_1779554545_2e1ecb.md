# 论证图


## YOLO系列算法在检测速度、精度和模型轻量化方面持续突破，成为最广泛使用的目标检测框架之一。

YOLO系列算法在检测速度、精度和模型轻量化方面持续突破，成为最广泛使用的目标检测框架之一。

**依据：**

- YOLOv1以45 FPS实现63.4% mAP，开创实时检测新纪元。

- YOLOv3在COCO上以51 ms推理时间实现57.9% AP50，达到当时速度和精度的最优平衡。

- YOLOv4在COCO上以65 FPS实现43.5% AP，大幅超越所有实时检测器。

**论证保证：**

- 在标准数据集上的速度与精度指标是衡量目标检测框架有效性的公认标准。

## 局限性与反驳

**反驳：** YOLOv1对密集小目标检测效果较差，定位精度不足。

## YOLO将目标检测视为端到端回归问题，实现了从两阶段到单阶段的范式转变。

YOLO将目标检测视为端到端回归问题，实现了从两阶段到单阶段的范式转变。

**依据：**

- 两阶段方法如R-CNN需要生成约2000个候选区域并逐一处理，计算冗余难以实时。

- YOLOv1通过单一CNN实现从图像像素到检测结果的端到端映射。

**论证保证：**

- 消除候选区域生成步骤可大幅减少计算冗余，从而提升检测速度。
