# 论证图


## 产业结构升级是大学生就业的核心驱动力

产业结构升级是大学生就业的核心驱动力

**依据：**

- 第三产业占比每提升1个百分点，就业率显著上升0.176%（t=3.25）

**论证保证：**

- 第三产业尤其是现代服务业具有显著的高学历劳动力偏好

  - （支撑）李兰冰（2021）测算表明，服务业占比每提升1%，大学生就业需求增长0.7%

## 局限性与反驳

**反驳：** 经济增长若依赖低端制造业或基建投资，反而加剧大学生“供需错配”

## 经济发展对就业的拉动存在双重门槛效应

经济发展对就业的拉动存在双重门槛效应

**依据：**

- 仅当服务业占比＞50%且人均GDP＞8万元时，经济增长的就业促进效应显著增强

## 产业结构对经济—就业关系具有正向调节作用

产业结构对经济—就业关系具有正向调节作用

**依据：**

- 交乘项系数0.045，且在东部地区表现尤为突出

**限定：** 产业结构对经济—就业关系的调节效应在东部地区表现尤为突出

## 中西部地区因产业低端化陷入“增长无效”困境

中西部地区因产业低端化陷入“增长无效”困境

**依据：**

- 异质性分析表明，中西部地区因产业低端化陷入“增长无效”困境

## 局限性与反驳

**反驳：** 部分中西部省份GDP增速领先但大学生失业率攀升，根源在于“重工业化路径”抑制高端服务业岗位供给
